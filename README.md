# 计算机网络名词解释连线练习（GitHub Pages版）

这是根据上传的《计算机网络期末复习习题集》整理的静态练习站点。

## 内容范围

- 只包含各章“名词解释”题。
- 不包含选择题。
- 不包含判断题。
- 共抽取 77 条名词解释。

## 本地打开

直接双击 `index.html` 即可练习。

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 把本文件夹里的 `index.html`、`style.css`、`app.js`、`data.js`、`README.md` 上传到仓库根目录。
3. 打开仓库 `Settings` → `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub Pages 自动部署。

## 使用建议

- 考前先按章节刷，再刷“全部章节”。
- 每轮 8-10 题最接近连线题的练习节奏。
- 批改后的错题会记录在浏览器本地，可以选择“只练错题”。
- “打印当前题”可打印一轮空白连线题。
